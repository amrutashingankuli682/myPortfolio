HairFall stage Classifier

The goal of this project was to develop a predictive model that can accurately determine the likelihood of hair loss based on various input features. By leveraging a Support Vector Machine (SVM) with a polynomial kernel, the model aimed to capture complex, non-linear relationships in the data that could influence hair loss. The project involved careful data preprocessing, including standardization, to ensure the model performed optimally. The outcome was evaluated using key metrics such as a confusion matrix and classification report, demonstrating the model's effectiveness.

Key Features:

  • SVM Model: Utilized a Support Vector Machine with a polynomial kernel for accurate hair loss prediction.
  • Data Standardization: Applied StandardScaler to normalize features for better model performance.
  • Model Evaluation: Assessed accuracy using a confusion matrix and classification report.
  • Training/Test Split: Implemented an 80-20 split to validate model performance on unseen data.
  • Predictive Insights: Identified key factors contributing to hair loss.
  • Tools Used: Employed Pandas, NumPy, and Scikit-learn for data processing and model development.

Technology Stack:

  • Pandas for data manipulation.
  • NumPy for numerical computations.
  • Scikit-learn for implementing the SVM model, data preprocessing, and evaluation.
