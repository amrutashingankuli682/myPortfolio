Virtual Try-On System

Key Features:

  • Real-Time Simulation: The system provides a seamless experience where virtual clothing adjusts to the user's body movements in real-time.
  • Body Tracking: Utilizes Mediapipe for precise body tracking, enabling accurate overlay of virtual garments.
  • Image Processing: OpenCV is used for efficient image processing, ensuring that the virtual try-on experience is smooth and realistic.
  • Responsive Design: The user interface is designed using HTML and CSS, ensuring compatibility across various devices.

Technology Stack:

  • Utilized OpenCV for image processing, Mediapipe for body tracking, and CSS and HTML for the user interface design.
